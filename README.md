## Getting Started

### Prerequisites

1. npm
  ```bash
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
  ```bash
  git clone https://github.com/sleepy-wood/blockchain-sample.git
  ```
2. Install NPM packages
  ```bash
  npm i
  ```
3. Run App
  ```bash
  node nodeJSCoin
  ```
